# 프로젝트 블루프린트

## 1. 프로젝트 개요
이 프로젝트는 웹 기반의 간단한 게임들을 개발하는 것을 목표로 합니다. 현재는 "Cosmic Fortune"이라는 이름의 "오늘의 운세" 게임이 구현되어 있으며, 사용자의 피드백을 반영하여 지속적으로 개선하고 있습니다.

## 2. 구현된 기능 및 디자인

### 2.1. 오늘의 운세 게임 (Cosmic Fortune)
사용자가 하루에 한 번 오늘의 운세 점수를 확인하고, 그 점수에 따라 격려의 메시지와 시각적인 피드백을 받는 웹 컴포넌트 기반 게임입니다.

*   **핵심 기술:** 순수 JavaScript, CSS, HTML을 사용하며, 재사용성을 위해 웹 컴포넌트(`FortuneTeller`)로 모듈화되어 있습니다. 외부 라이브러리에 대한 의존성이 없습니다.

*   **주요 기능:**
    *   **두 가지 운세 (금전운, 인간관계운):** '금전운'과 '인간관계운' 두 가지 독립적인 운세 점수(1-100점)와 그에 따른 메시지를 제공합니다.
    *   **하루 한 번 제한:** `localStorage`를 사용하여 사용자가 하루에 한 번만 운세를 확인할 수 있도록 제한합니다. 확인 후 버튼은 비활성화됩니다.
    *   **상태 유지:** 페이지를 새로고침해도 당일 확인했던 운세 점수와 메시지가 그대로 복원되어 표시됩니다.
    *   **동적 점수 애니메이션:** 운세 확인 시, 점수가 0부터 목표 점수까지 부드럽게 올라가는 애니메이션을 통해 시각적 즐거움을 더했습니다.

*   **"Bold" 디자인 시스템 적용:**
    *   **과감한 타이포그래피:** 미래적인 느낌의 'Orbitron' 폰트를 사용하여 제목과 본문 텍스트 간의 명확한 위계를 설정했습니다.
    *   **생동감 있는 색상 및 배경:** 오로라를 연상시키는 동적인 그라데이션 배경과 네온 색상의 'Glow' 효과를 통해 깊이감과 시각적 매력을 극대화했습니다.
    *   **입체적 카드 디자인:** 반투명한 'Glassmorphism' 효과와 다중 레이어 그림자를 적용하여 카드가 배경 위에 떠 있는 듯한 입체감을 연출했습니다.
    *   **직관적인 아이콘:** 재물(💰), 관계(🤝), 테마(☀️/🌙) 등 기능에 맞는 아이콘을 사용하여 사용자의 직관적인 이해를 돕습니다.
    *   **반응형 레이아웃:** 모바일, 태블릿, 데스크탑 등 모든 화면 크기에서 최적의 경험을 제공합니다.
    *   **라이트/다크 테마:** 사용자가 선호하는 테마를 선택하고 저장할 수 있는 토글 스위치를 제공합니다.

### 2.2. 최근 개선 사항
*   **버튼 스타일 버그 수정:** 웹 컴포넌트의 Shadow DOM으로 인해 외부 CSS 변수가 적용되지 않아 버튼이 깨져보이던 문제를 `main.js` 내부 스타일 시트에 변수를 직접 정의하여 해결했습니다.
*   **마크다운 렌더링 오류 수정:** 운세 결과 메시지에 포함된 마크다운 문법(`**`)이 일반 텍스트로 노출되던 버그를 HTML `<strong>` 태그로 교체하여 텍스트가 올바르게 굵게 표시되도록 수정했습니다.

## 3. 향후 개발 계획 (우리의 아이디어!)

사용자와의 논의를 통해 다음과 같은 멋진 기능들을 추가하기로 계획했습니다.

### 3.1. (최우선) 조명 애니메이션 복구
*   **문제:** 현재 배포된 버전에서 배경의 움직이는 조명(Glow) 효과가 작동하지 않고 있습니다.
*   **해결:** 이는 `main.js` 파일이 잘못된 버전으로 배포되었기 때문입니다. **가장 먼저 이 문제를 해결하여 앱의 핵심적인 시각적 매력을 되살립니다.**

### 3.2. 다국어 지원 (한/영 버전)
*   **목표:** 사용자가 한국어와 영어를 선택할 수 있는 기능을 추가하여 글로벌 사용자를 확보합니다.
*   **구현:**
    *   UI에 언어 선택 토글 또는 버튼을 추가합니다.
    *   `h1` 제목, 버튼 텍스트, 운세 카테고리(Wealth, Bonds), 운세 결과 메시지 등 앱 내의 모든 텍스트를 한글과 영문으로 제공합니다.
    *   사용자의 언어 설정을 `localStorage`에 저장하여 다음 방문 시에도 유지되도록 합니다.

### 3.3. 배경 콘텐츠 강화
*   **목표:** 현재 비어있는 배경 공간을 채워 더욱 풍성하고 인터랙티브한 사용자 경험을 제공합니다.
*   **구현:**
    *   **별가루 효과:** 마우스 움직임에 따라 반짝이는 별가루들이 은은하게 따라오는 인터랙티브 효과를 추가합니다.
    *   **클릭 가능한 별자리:** 배경에 여러 별자리들을 배치하고, 사용자가 클릭하면 해당 별자리의 이름이나 간단한 정보를 보여주는 기능을 추가합니다.
    *   **오늘의 우주 명언:** 운세 카드와는 별개로, 화면 한쪽에 매일 다른 우주나 과학과 관련된 명언을 표시하여 사용자에게 영감을 줍니다.
