# 프로젝트 블루프린트

## 1. 프로젝트 개요
이 프로젝트는 웹 기반의 간단한 게임들을 개발하는 것을 목표로 합니다. 현재는 "오늘의 운세"라는 게임이 구현되어 있으며, 사용자 인터랙션과 시각적 매력을 향상시키는 방향으로 개선되었습니다. 향후 새로운 게임을 추가할 예정입니다.

## 2. 구현된 기능 및 디자인

### 2.1. 오늘의 운세 게임 (Luck of the Day)
사용자가 하루에 한 번 오늘의 운세 점수를 확인하고, 그 점수에 따라 격려의 메시지와 시각적인 피드백을 받는 웹 컴포넌트 기반 게임입니다.

*   **웹 컴포넌트:** `main.js`에 `FortuneTeller`라는 이름의 커스텀 엘리먼트로 구현되어 있습니다.
*   **주요 기능:**
    *   **두 가지 운세 (금전운, 인간관계운):** 기존 단일 운세에서 '금전운'과 '인간관계운' 두 가지 독립적인 운세 점수를 제공합니다. 각 운세는 1-100점 사이의 랜덤 점수와 개별 메시지를 가집니다.
    *   **하루 한 번 운세 확인:** `localStorage`를 사용하여 사용자가 24시간 내에 한 번만 운세를 확인할 수 있도록 제한합니다. 버튼 상태가 '운세 보기'에서 '운세 확인 완료'로 변경되며 비활성화됩니다.
    *   **점수별 동적 메시지:** 1-100점 사이의 운세 점수에 따라 다른 내용의 격려 메시지와 이모지가 표시됩니다.
*   **디자인 및 스타일 (기존):**
    *   기본적인 HTML 구조와 최소한의 CSS 스타일링.

## 3. 현재 작업 계획: "Bold" 디자인 시스템 적용

사용자의 피드백("모양이 구리다")을 반영하여, 앱의 전체적인 디자인을 전문적이고 시각적으로 매력적이도록 대대적으로 개편합니다.

*   **목표:** 단순한 정보 제공을 넘어, 사용자가 앱을 사용하는 것 자체를 즐길 수 있는 세련되고 인상적인 사용자 경험(UX)을 제공합니다.

*   **적용될 "Bold" 디자인 원칙:**
    1.  **과감한 타이포그래피:** 영문 폰트 'Orbitron'을 사용하여 미래적이고 유니크한 느낌을 주며, 제목, 부제목, 본문 텍스트 간의 명확한 위계를 설정하여 가독성을 극대화합니다.
    2.  **생동감 있는 색상 팔레트 및 그라데이션:**
        *   **배경:** 깊이감 있는 다크 네이비(#1a202c)를 기본으로, 오로라를 연상시키는 동적인 그라데이션 배경을 추가하여 시각적 즐거움을 줍니다.
        *   **포인트 색상:** 생동감 있는 네온 퍼플, 핑크, 블루를 사용하여 상호작용 요소(버튼, 점수)에 활기를 불어넣습니다.
    3.  **깊이감과 입체감 (3D 효과):**
        *   **카드 디자인:** 반투명한 'Glassmorphism' 효과와 미묘한 그림자를 적용하여 카드가 배경 위에 떠 있는 듯한 입체감을 연출합니다.
        *   **그림자:** 다중 레이어 그림자와 빛나는 'Glow' 효과를 버튼과 중요 요소에 적용하여 깊이와 상호작용성을 강조합니다.
    4.  **아이콘과 시각적 요소:**
        *   재물, 관계, 테마 변경 등 각 기능에 맞는 세련된 아이콘을 사용하여 사용자의 직관적인 이해를 돕습니다.
    5.  **동적인 상호작용 및 애니메이션:**
        *   **테마 토글:** 단순한 체크박스가 아닌, 낮과 밤을 상징하는 아이콘이 포함된 세련된 스위치 디자인을 적용합니다.
        *   **점수 표시:** 운세 점수가 나타날 때 부드러운 카운트업 애니메이션을 추가하여 사용자의 기대감을 높입니다.
    6.  **반응형 레이아웃:** 모바일, 태블릿, 데스크탑 등 모든 화면 크기에서 최적의 사용성을 제공하도록 레이아웃을 유연하게 구성합니다.
    7.  **접근성(A11Y):** 모든 사용자가 불편함 없이 앱을 사용할 수 있도록 색상 대비, 키보드 네비게이션 등을 고려하여 설계합니다.

*   **구체적인 구현 단계:**
    1.  `style.css`와 `main.js`의 `<style>` 블록을 전면 수정하여 새로운 디자인 시스템을 적용합니다.
    2.  필요한 경우, `index.html`의 구조를 수정하여 디자인에 맞게 최적화합니다.
    3.  Google Fonts에서 'Orbitron' 폰트를 가져와 적용합니다.
    4.  테마 토글, 점수 애니메이션 등 동적 기능을 `main.js`에 추가/수정합니다.
