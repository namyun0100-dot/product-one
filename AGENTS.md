# 작업 규칙 (CosmicPaw)

## 필수 원칙
- 모든 블로그 글은 **한국어(KO) + 영어(EN) 버전**을 동시에 작성한다.
- 두 버전 모두 **정확한 위치에 저장**하고, `blog.html` 목록과 `sitemap.xml`에 **둘 다** 반영한다.
- 각 글의 `canonical`, `og:url`, `twitter` 메타, `ld+json`은 **해당 버전 URL**로 정확히 맞춘다.

## 주간 운세 키워드 규칙
- “이번 주 운세 키워드”는 **항상 다음 주 날짜 범위**로 작성한다.
  - 예: `2월 9–15일`처럼 표기
- 주간 글은 **미리 작성/예약 성격**으로 톤을 맞춘다.

## 작업 전 체크
- 작업 시작 전에 이 문서(AGENTS.md)를 먼저 읽고 준수한다.
